Class 8: F5 Advanced WAF 14.1
=============================

This series of lab exercises is intended to explain and demonstrate four key features of **F5 Advanced Web Application Firewall**.

This class covers the following F5 Advanced WAF 14.1 topics:

- Threat Campaigns
- Brute Force and Credential Stuffing Protections
- Credential Theft Protections
- Bot Protections

Expected time to complete: **3 hours**

.. WARNING::
    To ensure consistent behavior with the lab guide, **all lab exercises should be through the Chrome browser** on the
    Windows Jumphost.
